# World Time Converter

## 概要

**[World Time Converter](https://cloud8high.github.io/World-Time-Converter/)** は、2つの都市の日時を相互変換できるシンプルな Web アプリです。

片方の都市の日時を入力すると、もう一方の都市の日時がリアルタイムで表示されます。  
表示された2つの日時は、「`JST 2026年4月6日(月) 6:00 / EDT 2026年4月5日(日) 17:00`」形式のテキストとしてワンクリックでコピーできます。

ホスティング不要の静的サイトとして実装されており、GitHub Pages で公開しています。 → **[World Time Converter](https://cloud8high.github.io/World-Time-Converter/)** 

## 機能

- **2都市の日時を同時表示** — どちらの入力欄を変更しても、もう一方が自動で同期
- **都市の選択** — 東京・ロンドン・ニューヨーク・ロサンゼルスなど、世界20都市から選択可能
- **都市の入れ替え** — ⇄ ボタンで左右の都市を瞬時に入れ替え
- **UTC オフセット表示** — ドロップダウンに現在の UTC オフセット（例: `UTC+9:00`）を表示
- **ワンクリックコピー** — 「`JST 2026年4月6日(月) 6:00 / EDT 2026年4月5日(日) 17:00`」形式でクリップボードにコピー

## ポイント

### 夏時間 (DST) への完全対応

アメリカ・ヨーロッパ・オーストラリアなど夏時間を採用している地域のタイムゾーン変換を正確に処理します。

- ブラウザ標準の `Intl.DateTimeFormat` API を使用するため、**夏時間の切り替え日が変わっても自動対応**
- ドロップダウンの UTC オフセット表示も入力日時に連動して更新（例: ニューヨークは夏 `UTC-4:00` / 冬 `UTC-5:00`）
- タイムゾーン略称も夏時間を反映（例: `EDT` / `EST`）
